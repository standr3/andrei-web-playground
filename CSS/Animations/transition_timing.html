CSS transition-timing-function Demo

linear
Description: Constant speed from beginning to end.
Use when: For predictable, mechanical transitions (e.g., progress bars).
Common in: Sliders, timers.
ease (default)
Description: Starts slow, accelerates, then slows down.
Use when: You want a natural motion feel.
Common in: Buttons, image fades.
ease-in
Description: Starts slow, ends fast.
Use when: For entering motions or tooltip reveals.
Common in: Intro animations.
ease-out
Description: Starts fast, ends slow.
Use when: For exit motions or deceleration.
Common in: Closing modals, dropping objects.
ease-in-out
Description: Slow start and end, fast middle.
Use when: You want a balanced smooth motion.
Common in: Page transitions, scroll effects.
steps(10)
Description: Breaks motion into 10 discrete steps.
Use when: For retro, frame-by-frame, or sprite effects.
Common values: steps(2), steps(4, end), steps(10).
cubic-bezier(0.2, 0.75, 0.7, 0.25)
Description: Custom curve with full control over motion.
Use when: You need bounce, elastic, or advanced easing.
